# LogHelper
enable `qInfo qDebug qWarning etc.` sync into `.log` file

# QssHelper
load `qss` file

# How To Use
Just add one line `include(RELATIVE_PATH/qthelper.pri)` in the `.pro` file, such as `include(libs/qthelper/qthelper.pri)`. Use `git submodule add https://github.com/YourUser/QtHelper.git libs/qthelper` maybe better.
